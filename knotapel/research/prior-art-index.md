# Prior Art Index

One-line-per-paper quick reference across all knotapel literature surveys. Organized alphabetically by first author. Duplicates between surveys noted.

## Relevance Tags

- `[CORE]` — Directly foundational to DKC thesis
- `[BRIDGE]` — Connects two fields relevant to DKC
- `[CONTEXT]` — Provides theoretical context or methodology
- `[ADJACENT]` — Related work in adjacent territory
- `[TOPOLOGY]` — Pure knot theory / low-dimensional topology
- `[CVNN]` — Complex-valued neural networks
- `[LATTICE]` — Lattice codes / algebraic number theory
- `[ML-KNOTS]` — Machine learning applied to knot theory
- `[TQFT]` — Topological quantum field theory
